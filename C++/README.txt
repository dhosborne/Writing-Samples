Smart ATM Application



Compiling:

	All compilers must specify standard of C++11
	All file paths in Main are relative to Main.cpp; all included files are linked in Main.cpp.  No library linking necessary

	Windows - Using code blocks, ensure complier standard is C++11, build and run.

	Linux - Make file included.  Navigate to directory and type "make";  the "Main" executable will be created.



Run in windows:
	Click on Main.exe

Run in Linux:
	type "./Main"
